# The Thunderbolts

## Hintergrund

The Thunderbolts sind ein American-Football-Team, das derzeit an einer Saison teilnimmt. In dieser Saison gibt es verschiedene Bonuszahlungen für Spieler basierend auf ihrer Leistung. Die Spieler werden für verschiedene Ereignisse auf dem Spielfeld belohnt:

### Punkteboni:

*   **5 Punkte** für jeden erzielten Touchdown (TD)
*   **2 Punkte** für jedes erfolgreiche Field Goal (FG)
*   **1 Punkt** für jede Interception (INT)

### Zusatzboni und Abzüge:

Zusatzboni und Abzüge werden in der folgenden Reihenfolge beachtet:

1.  Spieler, die mehr als 3 TD in einem Spiel erzielen, erhalten einen Extra-Bonus von 10 % auf ihren Gesamtbonus.
2.  Spieler, die mindestens 1 INT und 1 FG erzielt haben, erhalten einen Zusatzbonus von 5 % auf ihren Gesamtbonus.
3.  Wenn ein Spieler mehr als 3 Fouls im Spiel gemacht hat, wird ein Abzug von 20 % vom Gesamtbonus vorgenommen.
4.  Wenn die Anzahl der erzielten Punkte (TD + FG + INT) insgesamt durch 7 teilbar ist, erhält der Spieler einen zusätzlichen Bonus von 50 Punkten.

- - -

## Aufgabe 1

Berechne den Gesamtbonus eines Spielers, der im Spiel 4 TD, 1 FG und 2 INT erzielt hat. Es gab keine Fouls.

- - -

## Aufgabe 2

Berechne den Gesamtbonus eines Spielers, der 3 TD, 2 FG und 1 INT erzielt hat und 4 Fouls begangen hat.

- - -

## Aufgabe 3

Berechne den Gesamtbonus eines Spielers, der 5 TD, 1 FG und 1 INT erzielt hat. Es gab keine Fouls.
